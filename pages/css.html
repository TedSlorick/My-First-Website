Beginner CSS

Basic Syntax

As a basic intro, CSS is essentially a list of rules which tells our HTML elements how they should be displayed. Each of these rule is made up of a few different components: selector, property, and value. 


            .button{
                background-color: white;
                padding: 10px
            }

Above is an example of what a CSS rule may look like, in this case; .button is the selector, background-color and padding are the properties, white and 10px are the values.

Selectors

I just mentioned selectors, but what are they? Selectors are the HTML elements which the CSS rules should be applied too. The available selectors include but is not limited to:

  • Universal Selector -
    *
    (affects the whole document)
  • Type Selectors -
    p
    (affects all elements with the p tag)
  • Class Selectors -
    .button
    (affects all elements with attribute class="button")
  • ID Selectors -
    #dog
    (affects all elements with attribute id="dog")

The major difference between ID and Class selectors is that any given HTML element can only have one class selector.
